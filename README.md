# mars-hotel-feedback

## Project Purpose

### Collect visitor feedback for the ```Lazy Spaceman Resort```.

## Technologies Used

1. XCode 12 & 13 - IDE
2. Swift 5 - Programming Language
3. UIKit/Storyboard
4. CoreData - Database
5. iOS 14.4 - IOS Version
